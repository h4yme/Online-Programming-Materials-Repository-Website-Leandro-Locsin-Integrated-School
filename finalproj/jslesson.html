JavaScript Learning Materials

MODULE 1: JAVASCRIPT
INTRODUCTION

JavaScript was invented by Brendan Eich in 1995 and became an ECMA standard in 1997. ECMA-262 is the official name of the standard. ECMAScript is the official name of the language.

MODULE 2: JavaScript Syntax, Statement, and Comments

A computer program is a list of "instructions" to be "executed" by the computer. In a programming language, these program instructions are called statements.

MODULE 4: JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task. These are blocks of code that can be named and reused. It is executed when "something" invokes it (calls it).

MODULE 3: JavaScript Variables, Operators, and Data Types

JavaScript variables are containers for storing data values.





MODULE 6: JavaScript Events and Strings

HTML events are "things" that happen to HTML elements. When JavaScript is used in HTML pages, JavaScript can "react" on these events.

MODULE 5: JavaScript Objects

You define (and create) a JavaScript object with an object literal. Spaces and line breaks are not important. An object definition can span multiple lines.

MODULE 7: JavaScript Numbers

In JavaScript, numbers are values that can be used in mathematical operations. You don’t need any special syntax for numbers — just write them straight into JavaScript (Example: 12345;)

MODULE 8: JavaScript Arrays

In JavaScript, an array is an ordered list of values. Arrays are container-like values that can hold other values. Each value is called an element specified by an index
